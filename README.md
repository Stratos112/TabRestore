# TabRestore
An incredibly simple tool for saving a broswer session and restoring all the tabs later. Like bookmarking except for contextual stuff that you might not care about next week, and you need information from multiple places. 

Chromium based. Will work on adding other support.

For now go to the extension manager, add from manifest. selenct manifest.json from this root.
